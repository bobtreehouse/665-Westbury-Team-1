USA Gas Price Tracker

Contructed By NYIT CSCI 665 Westbury Team 1 Fall 2022:

Robert Trieste, Alex Gusmano, Gummadi Pavani, Mohana Pranadeep Potti, Rajeev Gurram and Venkata Sundeep


Some highlights of this application:

  • Amazon AWS for the backend and service layers to host a Linux EC2 instance
  • Google Cloud Platform (GCP) to host and serve the dataframe produced
  • Microsoft Azure to host and run our Angular Javascript web app
  • Python and libraries like Beautiful Soup to scrape gas price data published by the American Automobile Association
  • Visual Studio as a cohesive IDE to build, debug and publish our code
  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling
